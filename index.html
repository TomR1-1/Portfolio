Website Portfolio (WIP)

Testing mobile display & responsivenes
Initial image sizing problem
Working on editing this modal
< >

This website is coded in HTML, CSS, and JavaScript with a little bit of jQuery. No ChatGPT or AI was used in creating this website! Prior to making this website, I had some web dev experience, making a photo storage system using the Django framework, but I primarily worked on the backend in Python. Furthermore, this is the first time I made this website mobile responsive.

The main challenge of this website was setting up the “My Projects” section. First, I had several issues regarding image sizing. It upset me how differently sized images looked uneven. I ended up encountering a solution to this problem after some time away on a social media app. It consisted of using aspect ratios, a maximum width, and making sure the image fit the container it was in while keeping the same aspect ratio (“object-fill: contain;”). Once I found this solution, it became a game of styling the page to my liking – this part is quite difficult itself without tools such as Figma.

My favourite part about my website is the pop-up modal that you are reading this text from. It required a lot of research and tuning, but it turned out great!

A key limitation I encountered is that this website needed to be static to host it on GitHub. This meant that despite having experience using the Django framework, I was not able to design a portfolio with it since it is a dynamic website. In the future I will consider this option.
